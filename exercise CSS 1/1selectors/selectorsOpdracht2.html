Selectors & Visual rules

opdracht 2

CSS pas je toe met een bepaalde syntax. Hoe ziet zo'n syntax eruit?
Kun je ook uitleggen wat elk element uit de syntax betekent?

antwoord:

CSS Syntax:

selector {eigenschap: waarde;}

bijvoorbeeld: h1 {color: blue;}

De seelector is in dit geval h1, deze wijst naar alle h1-tags in het document.
De eigenschap voor deze tags is de kleur, waar een waarde blauw aan wordt gegeven.
