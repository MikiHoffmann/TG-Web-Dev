antwoord:

cascade

Cascade is de volgorde waarin de styling in CSS wordt gelezen en uitgevoerd. Dit is van boven naar beneden.
BV: als je twee keer een selector gebruikt om een kleur te stylen zal de laatste gebruikt worden, die de eerdere overruled..

H3 met class cascade, kleur rood.

(twee keer een class selector onder elkaar, eerste kleur rood; de tweede kleur blauw, deze wordt derhalve gebruikt.)

inheritance

Inheritance is dat een style door child elements wordt overgenomen van de parent.
