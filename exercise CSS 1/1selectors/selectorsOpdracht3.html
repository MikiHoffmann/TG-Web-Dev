antwoord:

type selector

Een type selector selecteerd een bepaalde tag, zoals: <h3>

De <h3> tags in dit antwoord zijn met een "type selector" van kleur en lettertype veranderd.
Het voordeel is dat je alle gelijke tags in een document kunt veranderen.

descendant selector

De lijst hieronder is met een descendant selector van kleur veranderd.
  • type selectors
  • descendant selectors
  • class selectors
Het voordeel hiervan is dat je alle of bepaalde "children" van een tag kunt veranderen.
