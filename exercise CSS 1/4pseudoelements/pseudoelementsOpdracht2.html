Pseudo Elements

opdracht 2

Kun je enkele voorbeelden geven van pseudo-elementen (minimaal 4)?
Beschrijf met je eigen woorden wat deze pseudo-elementen inhouden.

antwoord:

::first line

wordt gebruikt om de eerste regel in eent tekst te beïnvloeden.

::selection

wordt gebruikt om een door de gebruiker geselecteerd gedeelte te beïnvloeden.

::slotted

wordt gebruikt om een element te selecteren dat in een template in een slot is geplaatst.

::part

wordt gebruikt om een element met een 'part-attribute' in een shadow DOM te selecteren. (templates)

::placeholder

::cue

::cue-region
