Box Model

opdracht 3

Wat gebeurt er als je een width en een height toevoegt aan de span tag ( ) van vorige opdracht?

antwoord:

De breedte van de inline-block wijzigd, waardoor er meer ruimte komt tussen de woorden.
De hoogte van de hele regel zal met de hoogte van de inline block veranderen.

Dit is een block box een neemt de hele breedte in waardoor de volgende box op de volgende regel zit

In dit block zit inline tekst in wat dus niet op de volgende regel komt te staan
