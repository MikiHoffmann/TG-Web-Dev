Welke position values zijn er? Leg voor elke position values uit wat het doet. Geef ook aan welke default position waarde elementen hebben die je gebruikt in html.

antwoord:

  2. relative
  3. fixed
  4. absolute
  5. sticky

Elementen zijn standaard in de static position en gaan met de normale 'flow'van de pagina mee.

Fixed betekend dat een element een vaste positie heeft t.o.v. de pagina randen en niet meescrollt.
